Send organized data to server.
Make data as **objects** in html using `fieldset` tag.
What does it means?
click [here](CASES.md) to see cases.

### Installing

`npm install --save fieldset-object`

### Examples

```javascript
const Fieldset = require('fieldset-object');

Fieldset.apply('form');
```
